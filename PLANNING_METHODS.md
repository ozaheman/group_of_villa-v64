# Villa Planning Methods - Quick Reference Guide

## Overview
This guide provides detailed information about the seven primary planning methods available in the villa layout generator. Each method has specific use cases, advantages, and trade-offs.

---

## 1. Grid Layout (Orthogonal)

### Description
Traditional urban pattern with rectangular blocks and straight roads intersecting at right angles.

### Best For
- Flat terrain
- High-density developments
- Efficient land use
- Urban areas with grid street systems

### Characteristics
- **Geometry**: 90° intersections
- **Block Size**: Regular rectangles
- **Navigation**: Simple, intuitive
- **Infrastructure**: Most economical (straight pipes and cables)

### Advantages
✅ Maximum land utilization  
✅ Lowest infrastructure cost per plot  
✅ Easy navigation and wayfinding  
✅ Predictable lot dimensions  
✅ Efficient service vehicle access  

### Disadvantages
❌ Can feel monotonous  
❌ Limited views/variety  
❌ Through-traffic encourages higher speeds  
❌ Less adaptable to topography  

### Recommended Parameters
- Plot Width: 12-15m
- Plot Depth: 20-30m
- Road Width: 9-12m
- Density: Medium to High

---

## 2. Cul-de-Sac Layout (Finger)

### Description
Main spine road with several dead-end streets branching off like fingers.

### Best For
- Family-oriented communities
- Suburban developments
- Safety-focused neighborhoods
- Privacy-seeking residents

### Characteristics
- **Geometry**: Main road + dead-end branches
- **Cul-de-sac Diameter**: Minimum 15m turning circle
-**Branch Length**: 60-120m typical
- **Traffic**: No through-traffic

### Advantages
✅ Enhanced privacy and safety  
✅ No through-traffic (safer for children)  
✅ Lower vehicle speeds  
✅ Creates mini-neighborhoods  
✅ Higher perceived property values  

### Disadvantages
❌ Traffic concentration at main spine  
❌ Longer routes for emergency vehicles  
❌ Reduced pedestrian connectivity  
❌ Higher infrastructure cost per plot  
❌ Turning around required  

### Recommended Parameters
- Cul-de-sac Diameter: 15-18m
- Branch Length: 60-100m
- Main Road Width: 12-15m
- Branch Road Width: 6-9m

---

## 3. Cluster Layout (Courtyard)

### Description
Houses grouped around shared common spaces like parks, pools, or motor courts.

### Best For
- Community-focused developments
- Maximizing green space
- Social interaction
- Compact development

### Characteristics
- **Cluster Size**: 8-16 units per cluster
- **Common Space**: 20-40% of cluster area
- **Setbacks**: Reduced from common space
- **Parking**: Shared or perimeter

### Advantages
✅ Strong sense of community  
✅ Maximized usable green space  
✅ Reduced road/infrastructure length  
✅ Semi-private outdoor areas  
✅ Shared amenity costs  

### Disadvantages
❌ Less individual plot frontage  
❌ Shared maintenance responsibilities  
❌ Closer proximity to neighbors  
❌ Parking management challenges  

### Recommended Parameters
- Cluster Size: 10-14 units
- Common Space: 300-500m²
- Plot Size: 150-250m²
- Green Space: 25-35%

---

## 4. Loop Layout

### Description
Roads curve around and connect back to themselves or main arteries, creating continuous circulation.

### Best For
- Balanced traffic flow
- Waste collection efficiency
- Gentle slopes
- Modern subdivisions

### Characteristics
- **Loop Perimeter**: 200-400m
- **Direction**: One-way or two-way possible
- **Intersections**: T-junctions preferred
- **Block Shape**: Irregular, organic

### Advantages
✅ Good traffic circulation  
✅ Efficient waste/delivery access  
✅ No dead-ends  
✅ Adaptable to terrain  
✅ Multiple route options  

### Disadvantages
❌ Can encourage higher speeds  
❌ More complex than grid  
❌ Larger turning radii needed  

### Recommended Parameters
- Loop Length: 250-350m
- Road Width: 9-12m
- Turning Radius: 15-20m
- Density: Medium

---

## 5. Linear Layout (Single-Loaded)

### Description
Villas arranged in a single line, usually facing a specific amenity or view.

### Best For
- Waterfronts
- Golf courses
- Ridge lines with views
- Premium developments

### Characteristics
- **Road**: One side only
- **Orientation**: All units face amenity
- **Plot Depth**: Flexible (often deep)
- **Frontage**: Premium pricing

### Advantages
✅ Every unit gets "the view"  
✅ Maximum privacy  
✅ Premium market positioning  
✅ Clear orientation  
✅ Unobstructed amenity access  

### Disadvantages
❌ Very high land use per unit  
❌ Expensive infrastructure (road serves one side only)  
❌ Limited density  
❌ Not suitable for flat, landlocked sites  

### Recommended Parameters
- Plot Width: 15-25m
- Plot Depth: 40-60m
- Road Width: 9-12m
- Density: Low

---

## 6. Organic Layout (Contour)

### Description
Layout follows natural topography, avoiding geometric shapes and respecting landforms.

### Best For
- Sloped terrain
- Ecologically sensitive sites
- Luxury resorts
- Unique character developments

### Characteristics
- **Roads**: Follow contour lines
- **Plot Shapes**: Irregular
- **Grading**: Minimal
- **Trees/Features**: Preserved

### Advantages
✅ Minimal environmental impact  
✅ Unique, character-rich design  
✅ Works with difficult terrain  
✅ Preserves natural features  
✅ Higher perceived value  

### Disadvantages
❌ Most expensive to develop  
❌ Complex surveying/plotting  
❌ Irregular lot boundaries  
❌ Challenging infrastructure routing  
❌ Variable plot sizes  

### Recommended Parameters
- Plot Size: Highly variable (200-500m²)
- Road Width: 6-9m (narrower acceptable)
- Max Slope: Follow 8-12% gradients
- Turning Radius: 12-15m

---

## 7. Radburn Layout (Pedestrian-Centric)

### Description
Separates pedestrian and vehicular traffic. Houses face a pedestrian green belt; garages/service roads at rear.

### Best For
- Sustainable communities
- Family-oriented developments
- Walkable neighborhoods
- Modern eco-villages

### Characteristics
- **Dual Access**: Front pedestrian, rear vehicular
- **Green Belt**: Central pedestrian spine
- **Garages**: Rear service lanes
- **Underpasses**: Pedestrians cross roads safely

### Advantages
✅ Extremely safe for walking/cycling  
✅ Park-like atmosphere  
✅ Low traffic exposure  
✅ Healthy lifestyle promotion  
✅ Strong community identity  

### Disadvantages
❌ Most expensive infrastructure  
❌ Requires large land area  
❌ Cultural shift needed (parking at back)  
❌ Maintenance of dual systems  

### Recommended Parameters
- Green Belt Width: 15-25m
- Service Lane Width: 5-7m
- Plot Width: 12-18m
- Plot Depth: 25-35m (to accommodate dual access)

## Hybrid Approaches

### Common Combinations

1. **Grid + Cul-de-Sac**
   - Grid for main roads
   - Cul-de-sacs for residential pockets
   - Best of both worlds

2. **Loop + Cluster**
   - Loop roads for circulation
   - Clusters within loops
   - Community + efficiency

3. **Organic + Radburn**
   - Follow contours
   - Pedestrian-friendly
   - Eco-resort style

## Implementation in Software

### Selecting Methods
1. Navigate to **Section 3: Layout Parameters**
2. Scroll to **Planning Method for Generative Design**
3. Check desired method(s)
4. Click **Generate Layout Solutions**

### Multiple Methods
- Check multiple boxes to generate variations
- System creates options using each selected method
- Compare results in solutions gallery

### Default Setting
- **Loop Layout** is checked by default
- Balanced approach for most sites
- Can be uncombined with others

## Tips for Optimal Results

1. **Match Method to Site**
   - Flat = Grid or Cluster
   - Sloped = Organic or Linear
   - Mixed = Loop

2. **Consider Market**
   - Budget buyers = Grid or Loop
   - Families = Cul-de-Sac or Radburn
   - Luxury = Linear or Organic

3. **Check Regulations**
   - Some jurisdictions prohibit cul-de-sacs
   - Others require grid connectivity
   - Fire access rules may dictate layout

4. **Test Multiple Methods**
   - Generate 2-3 options
   - Compare density, cost, aesthetics
   - Present to stakeholders
